# Atividade 3  Interatividade e Funcionalidades
Esta terceira entrega tem como foco a implementação de JavaScript avançado, transformando a interface estática em uma aplicação web dinâmica e interativa.
O objetivo é demonstrar domínio sobre manipulação do DOM, tratamento de eventos, armazenamento local e uso de templates, simulando o comportamento de uma aplicação real.
Manipulação do DOM
Implementação de um sistema básico de Single Page Application (SPA), permitindo troca dinâmica de conteúdo sem recarregar a página.
Criação e uso de templates JavaScript para renderização modular das seções do site (Início, Projetos, Sobre, Contato e Cadastro).
Estrutura do Projeto
Código organizado por pastas e módulos, incluindo:
/js → arquivos JavaScript modulares
/imagens → recursos visuais
/videos e /audios → mídia institucional
/css → folha de estilos principal (styles.css)
index.html → página principal e ponto de entrada da aplicação
uncionalidades Implementadas
Menu hambúrguer responsivo, funcional em desktop e mobile;
Navegação dinâmica entre páginas (SPA) usando JavaScript modular;
Seções de Galeria de Imagens, Vídeo Institucional e Áudio Institucional;
Validação de formulários com avisos de preenchimento incorreto (máscaras de CPF, telefone e CEP);
Estrutura acessível com uso de ARIA e boas práticas de semântica HTML;
Sistema pronto para publicação via GitHub Pages.
Tecnologias Utilizadas
HTML5 → Estrutura e semântica
CSS3 → Estilização e responsividade
JavaScript (ES6 Modular) → Lógica, interatividade e templates
Git e GitHub → Versionamento e hospedagem
